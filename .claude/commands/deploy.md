# Deploy Command

Before deploying:

1. Run all tests and ensure they pass
2. Check for any console.log or debug code
3. Verify environment variables are set
4. Build the project
5. Run database migrations if needed
6. Deploy

## Checklist
- [ ] Tests passing
- [ ] No debug code
- [ ] Environment variables set
- [ ] Build successful
- [ ] Database migrated
- [ ] Deployment complete
